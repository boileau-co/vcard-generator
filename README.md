# vCard Generator

WordPress plugin — manage employee contact records and serve them as downloadable `.vcf` files at clean public URLs, with QR code generation and scan tracking.

## Requirements

- WordPress 6.4+
- PHP 8.1+
- Composer (for initial setup)

## Setup

```bash
composer install --no-dev --optimize-autoloader
```

Then activate the plugin in WP Admin. Visit **vCards → Settings** to configure organization defaults before creating vCards.

## Public URL

```
https://example.com/v/{slug}
```

The slug base (`v`) is configurable in Settings → vCard Generator → URL Configuration. Changing it flushes rewrite rules automatically.

## vCard format

vCard 3.0, UTF-8, CRLF line endings, RFC 6350 line folding for lines over 75 bytes. Empty fields are omitted. Phone numbers are normalized to E.164 on save.

## QR codes

Generated by `chillerlan/php-qrcode` (MIT). SVG is the primary format; PNG (1024×1024) is the fallback. Quiet zone is always 4 modules. Error correction level defaults to M (15%) and is configurable per-site in Settings.

## Scan tracking

When enabled, each hit on `/v/{slug}` increments `_vcard_generator_scan_count` and updates `_vcard_generator_last_scanned`. HEAD requests and known bot user agents are excluded. No personal data is stored. The bot list is extensible via the `vcard_generator_bot_user_agents` filter.

## Hooks

| Hook | Type | Description |
|---|---|---|
| `vcard_generator_bot_user_agents` | filter | Array of bot user-agent substrings to skip when counting scans |
